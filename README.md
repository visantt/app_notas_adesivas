# App de Notas Adesivas

Aplicação web para criação e gerenciamento de notas adesivas, com foco em organização de informações de forma simples e eficiente.

---

## Funcionalidades

* Criação de notas
* Edição de conteúdo
* Remoção de notas
* Persistência de dados com LocalStorage
* Tela de cadastro de usuário
* Dashboard para gerenciamento

---

## Tecnologias

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## Objetivo

Este projeto foi desenvolvido para praticar conceitos fundamentais de desenvolvimento web, incluindo:

* Manipulação do DOM
* Organização de aplicações com múltiplas páginas
* Estruturação de código JavaScript
* Armazenamento local de dados
